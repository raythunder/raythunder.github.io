gulp实例

2017-09-07 15:18:51

# 新建gulpfile.js文件

根据实际使用注释掉了一些不需要的插件

# 初始化环境

npm init  # 生成package.json

# 安装插件

根据gulp.js安装所需插件

npm install --save gulp gulp-plumber gulp-less browser-sync gulp-useref gulp-uglify gulp-if gulp-cssnano gulp-cache del run-sequence gulp-autoprefixer gulp-sourcemaps lazypipe

# 目录结构

cmd 进入某文件夹，输入 tree /F 可生成文件目录树（win）

│  gulpfile.js
│  package.json
│
└─src
    │  index.html
    │
    ├─css
    │      style.css
    │
    ├─img
    ├─js
    │      index.js
    │
    └─less
            style.less

# 开发运行

gulp   # 在根路径运行，此时是开发环境，浏览器自动打开，less实时编译，页面自动刷新。

# 打包

将页面中需要打包的文件放入特定的注释中

<!DOCTYPE html>
<html lang="en">
<head>
	<meta charset="UTF-8">
	<title>Document</title>
	<!--build:css css/style.min.css -->
	<link rel="stylesheet" type="text/css" href="css/style.css">
 	<!-- endbuild-->	

	<!--build:js js/index.min.js -->
    <script type="text/javascript" src='js/index.js'></script>
 	<!-- endbuild-->	
</head>

gulp build  # 打包生成文件到dist目录

生成结果：

│  gulpfile.js
│  package.json
│
├─dist
│  │  index.html
│  │
│  ├─css
│  │      style.min.css
│  │
│  ├─js
│  │      index.min.js
│  │
│  └─maps
│      ├─css
│      │      style.min.css.map
│      │
│      └─js
│              index.min.js.map
│
└─src
    │  index.html
    │
    ├─css
    │      style.css
    │
    ├─img
    ├─js
    │      index.js
    │
    └─less
            style.less
